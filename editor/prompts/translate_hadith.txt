Распознай перевод Хадиса в следующий json:
{
    "arabic_fragment": {
        "text": "арабский текст цитаты",
        "source": "арабское название книги и номер хадиса или несколько книг и хадисов",
    },
    "russian_fragment": {
        "lines": [
            "русский текст строка 1",
            "русский текст строка 2",
        ],
        "source": "русскоеназвание книги и номер хадиса или несколько книг и хадисов",
    },
    "table": [
        {
          "arabic": "фрагмент текста построчного перевода на арабском",
          "russian": "фрагмент текста построчного перевода на русском",
        },
        ...
    ]
}

Если в тексте нет названия книги на арабском или на русском, найди его и добавь - больше ничего не меняй!
Кроме названий книг не дописывай никакие поля сам, если контент для них отсутствует в моем тексте, оставь их как null.
Не придумывай и не ищи переводов самостоятельно, все по моему тексту.
Русский текст разбей на строки russian_fragment.lines - где каждая строка это либо прямая речь: "Передал Абу Нуайм, что пророк сказал" - либо не прямая речь: "Проклятие Аллаха и взяточнику, и дающему взятки!"
Русское название книги пиши без кавычек, в формате "Сахих Аль-Бухари 6984"
Ни в коем случае не изменяй мой текст!
В поле table должен быть построчный перевод - если он есть в моем тексте, если нет - оставь table: null.
Если нет арабского текста - оставь arabic_fragment: null
Если нет русского текста - оставь russian_fragment: null
Верни результат в json - не как цитата через ```json - просто в формате для парсинга
