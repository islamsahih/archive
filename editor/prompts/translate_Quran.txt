Распознай перевод цитаты Корана в следующий json:
{
    "arabic_fragment": {
        "text": "арабский текст цитаты",
        "source": "арабское название суры и номер аята",
    },
    "russian_fragment": {
        "text": "русский текст цитаты",
        "source": "русское название суры и номер аята",
    },
    "table": [
        {
          "arabic": "фрагмент текста построчного перевода на арабском",
          "russian": "фрагмент текста построчного перевода на русском",
        },
        ...
    ]
}

Если в тексте нет названия суры на арабском или на русском, найди его и добавь - больше ничего не меняй!
Кроме названий сур не дописывай никакие поля сам, если контент для них отсутствует в моем тексте, оставь их как null.
Не придумывай и не ищи переводов самостоятельно, все по моему тексту.
Русское название суры пиши без кавычек, в формате "Сура Аль-Бакара, аят 16"
Ни в коем случае не изменяй мой текст!
В поле table должен быть построчный перевод - если он есть в моем тексте, если нет - оставь table: null.
Если нет арабского текста - оставь arabic_fragment: null
Если нет русского текста - оставь russian_fragment: null
Верни результат в json - не как цитата через ```json - просто в формате для парсинга
