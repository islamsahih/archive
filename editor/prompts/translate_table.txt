Распознай перевод c арабского на русский в следующий json:
{
    "table": [
        {
          "arabic": "фрагмент текста построчного перевода на арабском",
          "russian": "фрагмент текста построчного перевода на русском",
        },
        ...
    ]
}

Ни в коем случае не изменяй мой текст, ни арабский, ни русский!
Не придумывай и не ищи переводов самостоятельно, все по моему тексту.
В русском тексте могут быть комментарии, не являющиеся непосредственно переводом - они должны остаться в соответсвующем элементе table.
